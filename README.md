
# Restaurant Management System

A complete system for managing an Arabic restaurant, designed to simplify daily operations.

## Features

* Menu and items management
* Orders and sales system
* Discounts system
* Daily reports
* Data backup

## Installation

1. Upload all project files to the server or keep them locally
2. Open the `index.html` file directly in your web browser

## Important Notes

* This project relies on the **Web Storage API (window.localStorage / window.sessionStorage)** to store data.
* No backend or external database is required.
* The system runs using **a single HTML file only**, and works directly in the browser.


# نظام إدارة المطعم

نظام متكامل لإدارة مطعم عربي يشمل مجموعة من الوظائف الأساسية لتسهيل الإدارة اليومية.

## المميزات

* إدارة القائمة والأصناف
* نظام الطلبات والمبيعات
* نظام الخصومات
* تقارير يومية
* نسخ احتياطي للبيانات

## التثبيت

1. تحميل جميع الملفات على السيرفر أو على جهازك المحلي
2. فتح ملف `index.html` مباشرة في المتصفح

## ملاحظات مهمة

* هذا المشروع يعتمد على **Web Storage API (window.localStorage / window.sessionStorage)** لحفظ البيانات.
* لا يحتاج إلى Backend أو قاعدة بيانات خارجية.
* يوجد **ملف تشغيل واحد فقط (HTML)**، ويتم تشغيل النظام مباشرة من المتصفح.
